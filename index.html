Gauquelin5 auxiliary data

Auxiliary files of program https://github.com/tig12/gauquelin5 (noted g5).
These files are not necessary to run g5, but can be useful to check g5 or prepare future developments.
Contains (not exhaustive) :
  • Scans of paper documents to OCR
  • Scans of paper documents already OCRed
  • Files used to build g5 code or data
  • Development notes
